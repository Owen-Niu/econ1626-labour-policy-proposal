# ECON1626 Labour Policy Proposal

This repository contains my Assessment 2 policy proposal for ECON1626. The proposal examines how artificial intelligence is affecting the Australian labour market, including employment, tasks, wages, productivity, and distributional outcomes.

The policy brief evaluates three possible responses and recommends an Australian AI Workforce Compact that combines sectoral reskilling partnerships with an employer skills guarantee.

## Repository Structure

- `proposal.md` - the main 1,200-word policy proposal.
- `README.md` - an overview of the repository and how to navigate it.

## How to Read the Proposal

Open `proposal.md` on GitHub to view the rendered Markdown version of the policy proposal. The document includes:

- Executive Summary
- Problem Definition and Context
- Analysis of Impacts
- Policy Options and Evaluation
- Recommended Package and Implementation Plan
- Risks and Mitigations
- References
